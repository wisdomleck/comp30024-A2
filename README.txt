All the relevant files for our submission is located in:

skeleton-code-B/grandMasters

To run our program against another module, cd into skeleton-code-B directory and call:
python -m referee "grandMasters/player.py" <lower_module>


To run our program against another player on battleground, cd into battleground and call:
python -m battleground grandMasters <any username>


To run our program against a special player on battleground, cd into battleground and call:

python -m battleground grandMasters <any username> greedy
python -m battleground grandMasters <any username> random
